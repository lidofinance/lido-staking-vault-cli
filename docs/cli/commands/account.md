# Account

## Command

```bash
lsv-cli account [arguments] [-options]
```

## Account commands list

```bash
lsv-cli account -h
```

## API

| Command      | Description                                                                         |
| ------------ | ----------------------------------------------------------------------------------- |
| info         | general account info.                                                               |
| generate-key | generate a new key. Disclaimer: this command is not recommended for production use. |
